# 🎯 نظام توليد الدعوات

موقع بسيط لتوليد دعوات إلكترونية بشكل جماعي.

## 🚀 الرابط

بعد رفع الملفات على GitHub Pages:
```
https://ahmadcloud.github.io/invite
```

## 📁 الملفات

| الملف | الوظيفة |
|-------|---------|
| `index.html` | الواجهة الرئيسية |
| `style.css` | التصميم والألوان |
| `script.js` | المنطق البرمجي |
| `assets/background.png` | خلفية الدعوة |
| `assets/logo.png` | شعار المركز |

## 🎨 كيف تعدّل الخلفية

1. افتح [Photoshop / Illustrator / Canva]
2. أنشئ ملف بمقاس **1080 × 1920 بكسل**
3. صمّم الخلفية (اترك مساحة فارغة للنص)
4. احفظها باسم `background.png`
5. ارفعها في مجلد `assets`

## 🔤 كيف تغيّر الخط

1. افتح `index.html`
2. ابحث عن Google Fonts link
3. أضف الخط الجديد
4. افتح `style.css` وغيّر `font-family`

## 🌈 كيف تغيّر الألوان

افتح `style.css` وعدّل:
```css
:root {
    --accent: #c9a84c;        /* لون التمييز */
    --primary: #1a1a2e;       /* اللون الأساسي */
}
```

## ⚡ الاستخدام

1. افتح الرابط
2. ارفع الشعار (اختياري)
3. املأ بيانات المناسبة
4. أضف قائمة الأسماء
5. اضغط "توليد كل الدعوات"
6. حمّل ملف ZIP
